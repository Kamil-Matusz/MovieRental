# MovieRental
Project representing a movie rental shop with order handling made in .NET MVC 

### Technologies
 - C# 11
 - .NET 5.0
 - EF Core
 - Microsoft SQL Studio
 - .NET JWT
 - HTML
 - CSS
 - JavaScript
 - Bootstrap

# Theme
``Solar Theme``
